# Capture The Flag
Those Sites are Created for Educational Purpuses only, and for CTF Challanges.
None of shown websites are Working, or sending any data to 3rd party sites.

## Where/how to start
Visit this main site [here](https://miskajurostudios.github.io/CTF/sites/).
Find your desired challange, try everything,
Look at the Page Source, Javascripts, Webhooks, Hidden Elements, etc...
Sites are located in 'sites/challange/(challange number)/'
If you want to learn more about specific challanges,
or your lost visit the 'Docs' folder.

## Flag Format
Flags will be hidden at the end of the challange,
flag format is **MJS_Flag{random-junk00-here00}**
If you have problems with finding the flags, please
read 'Docs/Answers/Chalange-(challange number).md'
